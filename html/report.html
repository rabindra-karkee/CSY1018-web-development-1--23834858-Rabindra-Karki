Experience of learning coding

It was a fun adventure that required a lot of hard effort and perseverance to learn
the fundamentals of web development, code, design, and debugging. Understanding HTML required knowing how to structure web pages with tags and components as the first step. It was first fairly difficult, but the more the learner practiced, the simpler it became.

Understanding the principles of HTML and CSS at first was difficult and perplexing. The student found it difficult to design webpages that were visually appealing and had trouble understanding the fundamental ideas. The learner did, however, start to advance with perseverance and regular practice.

Problems and challenges

Debugging code posed one of the biggest difficulties faced while learning. It took time and careful attention to spot mistakes and repair them.

The student frequently discovered oneself wasting hours looking for a tiny fault in their code. Yet as time went on, debugging got more relaxed.
